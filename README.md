# 📚 Library Management System (C++)

## 📌 Description

This project is a simple Library Management System developed in C++ as part of my Programming Fundamentals course.
It demonstrates core programming concepts such as file handling, functions, and conditional logic.

---

## 🚀 Features

* Add new books
* Display all book records
* Search for a specific book
* Delete book records
* Store data using file handling

---

## 🛠 Technologies Used

* C++
* File Handling

---

## ▶️ How to Run

1. Open the project in any C++ IDE (e.g., VS Code or Dev C++)
2. Compile the program using:

   ```
   g++ main.cpp -o library
   ```
3. Run the program:

   ```
   ./library
   ```
